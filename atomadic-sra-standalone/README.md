# Atomadic SRA Standalone v3.2.1.0

The Supreme Research Agent (SRA) architecture, now available as a standalone, autonomous system.

## Setup

1. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   ```

2. **Initialize**

   ```bash
   python activate_agents.py
   ```

3. **Run Main Interface**
   Open `main.ipynb` in your preferred notebook environment (VS Code, Jupyter, Antigravity).

## Architecture

See `sra-blueprint.md` for the full architectural specification.

## Verification

Run `python verify_blueprint.py` to check system integrity.
